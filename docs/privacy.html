Privacy Policy

Last updated: July 2026

StudySouk Timer (the “Activity”) is a Discord Activity that provides a shared Pomodoro timer and social word games inside Discord voice channels. This policy explains what information the Activity uses and how.

1. Information we use

When you launch the Activity, we use only the minimum data needed to make it work:

  • Discord user ID and display name — obtained through Discord OAuth2 with the identify scope only, so we can show your name to the other players in the same room and track whose turn it is in a game.
  • In-session gameplay data — your current room, game state (lives, scores, turns) and the chat messages you send inside a room.

We do not request your email, friends list, messages outside the Activity, payment information, or any other Discord data.

2. How the data is used

  • To display player names and run the real-time shared games and timer.
  • To relay chat messages between players inside the same room.

We never sell your data, use it for advertising, or share it with third parties.

3. Storage and retention

Game and chat data is kept only in the server’s memory while a session is active. It is not written to any database or disk, and it is automatically discarded when a room becomes empty or when the server restarts. We do not keep long-term records of your activity.

4. Third parties

The Activity runs through Discord’s infrastructure and is subject to Discord’s Privacy Policy. Word-game dictionaries are static open-data word lists and do not involve any personal data.

5. Children

The Activity is intended for users who meet Discord’s minimum age requirements. We do not knowingly collect data from anyone below that age.

6. Your choices

Since we store nothing persistently, closing the Activity removes your data from the current session. To request any information, contact us below.

7. Changes

We may update this policy; the “Last updated” date above will reflect any change.
